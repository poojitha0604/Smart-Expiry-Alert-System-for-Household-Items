## Project Overview

The **Smart Expiry Alert System for Household Items** is a full-stack web application designed to help users track the expiry dates of everyday household products such as medicines, groceries, cosmetics, and other essential items.

The system allows users to store product details and expiry dates in a centralized database so they can easily manage and monitor items before they expire. This helps reduce wastage and improves household management.

This project demonstrates the implementation of a complete web application including frontend, backend, database integration, and deployment.

---

## Features

* User login system
* Add household items with expiry dates
* Store and manage items in a database
* Dashboard to track all stored products
* Simple and user-friendly interface
* Helps avoid using expired products

## How the System Works

1. Users log into the system.
2. Users add household items along with their expiry dates.
3. The information is stored in the MySQL database.
4. Users can view and manage their stored items through the interface.
5. The system helps users monitor products before they expire.
